# ASL Models

Repositorio de modelos de inferencia de lenguaje de senas empaquetados en contenedores Docker.

## Proyectos

- [DTW](DTW/README.md): entrenamiento, evaluacion e inferencia para el modelo DTW/SVM.
- [Transformer](transformer/README.md): entrenamiento y evaluacion de un modelo Transformer con PyTorch.
- [Mix Atencion](mix%20atencion/README.md): entrenamiento y evaluacion de modelos TensorFlow con atencion.
